# SVIL Baduk PRD (v0.1)

## 목표

저시력 사용자가 바둑을 **배우고 / 혼자 두고 / 친구와 두기** 할 수 있는 고대비 클라이언트.

## 핵심 요구

1. **단계별 배우기** — 판·돌·따냄·패·끝내기 레슨
2. **혼자서 두기** — 9/13/19, 급·단 선택, 내장 AI / KataGo
3. **상대랑 두기** — 서버리스 WebRTC P2P (PeerJS 시그널)
4. **고대비 UI** — 큰 교차점, 착수점 깜빡임, 키보드 착수, 직전 수 마커
5. **오픈 로컬 AI** — KataGo GTP (데스크톱 사이드카 예정)
