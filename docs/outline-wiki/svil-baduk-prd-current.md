# SVIL Baduk PRD

**버전:** 0.1.0  
**작성일:** 2026-07-23  
**상태:** 초안 (MVP 반영)

---

## 1. 제품 요약

SVIL Baduk은 **저시력 사용자**가 바둑을 배우고, AI와 연습하고, 친구와 서버 없이 둘 수 있도록 만든 **고대비 바둑 클라이언트**다.

- 브랜드/제품명: **SVIL Baduk**
- 플랫폼(현재): 웹 (Vite + React). 이후 데스크톱 셸(Tauri/Electron)로 KataGo 실연동
- 디자인: SVIL 고대비 다크 + 교보손글씨2019 폴백

## 2. 문제 · 기회

일반 바둑 앱은 목판 질감·작은 교차점·색만으로 흑백을 구분하는 UI가 많아 저시력 사용자에게 불리하다.  
학습·연습·친선 대국을 **한 앱**에서, **시각 대비를 최우선**으로 제공하면 SVIL 접근성 제품군과 잘 맞는다.

## 3. 목표 사용자

| 구분 | 설명 |
|------|------|
| 1차 | 저시력·고령 바둑 입문자/취미 대국자 |
| 2차 | 고대비·큰 UI를 선호하는 일반 사용자 |
| 환경 | Windows 데스크톱 브라우저 우선, 키보드·큰 터치 타겟 |

## 4. 제품 목표

1. 규칙을 **단계별로** 익힐 수 있다.
2. 급·단을 골라 **혼자서** AI와 둔다 (내장 AI → KataGo).
3. **상대와** 서버리스 P2P로 둔다.
4. 착수점이 크고 깜빡이며, 색+텍스트+포커스로 상태를 알 수 있다.
5. 오픈 로컬 AI(KataGo)를 쓸 수 있는 구조를 유지한다.

## 5. 핵심 요구사항

### 5.1 단계별 배우기
- 레슨: 판·돌 → 따내기 → 패 → 끝내기/집
- 각 단계 제목·본문·힌트, 이전/다음, 목차 점프
- (이후) 사활·따냄 연습 문제 보드

### 5.2 혼자서 두기
- 판: 9 / 13 / 19
- 상대 급·단: 30급 ~ 5단
- 내 색깔: 흑/백
- AI: 내장 휴리스틱 → KataGo GTP (연결 시)
- 패스·기권·직전 수 표시·포획 수

### 5.3 상대랑 두기
- 서버리스 WebRTC P2P (PeerJS 시그널만 사용)
- 방 ID 생성·복사·참가
- 호스트가 판 크기·색깔 결정, 게스트 동기화
- 수·패스·기권 메시지 동기화

### 5.4 고대비 · 접근성
- 다크 고대비 토큰, 본문 ≥16px(설정 16/18/20), 타겟 ≥50px
- 착수 가능 교차점 깜빡임 (움직임 줄이기 옵션)
- 키보드: 화살표/Tab·Enter 착수, Alt+← 홈
- 흑/백 돌에 텍스트 라벨, 직전 수 마커
- 글꼴 8종·언어 5종·설정 내 히스토리

### 5.5 로컬 AI
- KataGo 바이너리 + 모델 로컬 배치
- GTP transport 추상화 (`setKataGoTransport`)
- 미연결 시 내장 AI 폴백 및 상태 문구

## 6. 비목표 (현 버전)

- 자체 매치메이킹·계정·레이팅 서버
- 정식 중국/일본/한국 계가·덤 규칙 완성
- 모바일 스토어 배포
- 온라인 기보 DB·방송

## 7. 성공 지표 (정성)

- 저시력 사용자가 “어디서 두는지”를 착수점 깜빡임·포커스로 파악
- 입문자가 레슨만으로 따냄·패 개념을 설명 가능
- 두 브라우저에서 방 ID만으로 대국 성립
- KataGo 연결 시 급단별 체감 난이도 차이

## 8. 리스크
